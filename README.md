# Nile University Landing Page Clone

A responsive clone of the Nile University landing page built using HTML, CSS, and Bootstrap 5. This project was created to demonstrate frontend development skills, responsive web design, and the ability to recreate modern website layouts.

## Features
- Responsive navigation bar
- Modern landing page design
- Hero section with attractive layout
- Course information sections
- Bootstrap 5 responsive components
- Clean and structured user interface

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
